# Loop – Entwicklungszyklen

Die Entwicklung erfolgt iterativ. Jeder Zyklus umfasst folgende Schritte:

1. **Planung**: Definiere die nächsten Aufgaben anhand der Roadmap.
2. **Umsetzung**: Entwickle Features oder behebe Bugs.
3. **Code‑Review**: Lasse Änderungen durch ein Teammitglied oder automatisches Tool prüfen.
4. **Testing**: Schreibe und führe Tests durch, sowohl manuell als auch automatisiert.
5. **Deployment**: Deploye die Änderungen auf eine Testumgebung oder GitHub Pages.
6. **Feedback**: Sammle Feedback von Testnutzern und Stakeholdern.
7. **Analyse**: Bewerte die Ergebnisse (Fehler, Performance, UX) und passe den Plan an.

Wiederhole diese Schritte, um kontinuierlich zu verbessern.
