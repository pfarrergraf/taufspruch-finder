# MCP – Minimum Creative Product

Der Minimum Creative Product (MCP) definiert die kleinste funktionsfähige Version des Taufspruch Finder.

## Funktionale Anforderungen
- Auswahl von Lebenseinstellungen und Charaktereigenschaften.
- Anzeige von Bibelversen in mehreren Übersetzungen.
- Favoritenliste lokal im Browser.
- Responsives Design für Smartphones.

## Nicht-funktionale Anforderungen
- PWA-Fähigkeit mit Offline-Funktion.
- Ladezeiten < 2 Sekunden auf Mobilgeräten.
- Einhaltung der Datenschutzverordnung (DSGVO).

## Erweiterungen nach MCP
- Semantische Suche mit Vektor‑Datenbanken.
- Mehrsprachigkeit über Deutsch hinaus.
- Nutzerkonten und serverseitige Speicherung.
- KI‑gestützte Auslegungen und Empfehlungen.
